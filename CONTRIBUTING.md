# Contributing to Job Scout Engine

First off, thank you for considering contributing to Job Scout Engine! It is people like you who make this tool so powerful and helpful for job seekers.

## 🚀 How Can I Contribute?

### Reporting Bugs
*   Check the Issues tab to see if the bug has already been reported.
*   If not, open a new issue. Include a clear title, steps to reproduce, expected behavior, and actual behavior.

### Suggesting Enhancements
*   Open an issue explaining the feature suggestion and why it would be valuable.

### Pull Requests
1.  Fork the repository and create your branch from `main`.
2.  Install dependencies: `pip install -r requirements.txt`.
3.  Implement your changes, keeping coding style clean and documented.
4.  Ensure that linting and unit tests pass.
5.  Submit a pull request with a detailed description of your changes.

## 🛡️ Security Vulnerabilities
Please do not report security vulnerabilities (like leaked API keys or credentials) via public issues. Instead, contact the maintainers directly or submit a private report.
